		Ack(0, n) = n+1  
		Ack(m, 0) = Ack(m-1, 1), m > 0  
		Ack(m, n) = Ack(m-1, Ack(m, n-1)), m > 0, n > 0.		
Пример:		
ack(0,3) = 4
ack(0,4) = 5
ack(1,0) = 2
ack(1,1) = 3
ack(2,0) = 3
ack(2,1) = 5
